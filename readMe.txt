>> The app is build for Android Phones and implemeted in Java and XML.
>> No permissions requires.
>> Implementation of OnClickListener and OnLongClickListener functions on a button for RecyclerView.
>> OnLongClickListener is implemented with Contextual Action Mode (Read Below).
 

Contextual Action Mode:
	- is similar to floating context menu and appears when we long click on a menu.
	- But instead of showing options in a pop-up menu, they appears under app's action bar (Top most part where app name is shown).
